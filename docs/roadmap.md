# Roadmap

## MVP (v0.1)
- Experiments CRUD (минимум)
- Import CSV/JSONL
- Channels stats (min/max/avg/count)
- Series (для графиков)
- Summary (experiment)
- Export CSV
- Документация (TZ/arch/api/tests/decisions/defense)

## v0.2
- UI: страница эксперимента с выбором каналов и графиком
- История импортов по эксперименту
- Пресеты каналов (например "Температуры", "Давления")

## v0.3
- PostgreSQL/Timescale
- Downsampling (например: агрегация по минутам)
- Сравнение 2 экспериментов

## v1.0
- Live ingest (API/stream)
- Аномалии (threshold/rules)
- Роли/авторизация
