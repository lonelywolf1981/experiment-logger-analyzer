# Experiment Data Logger & Analyzer (MVP)

Внутренний инструмент для лаборатории/инженеров: импорт экспериментальных данных (CSV/JSONL), хранение, просмотр каналов, графики, сводка по эксперименту, экспорт.

## Зачем
- Собирать и хранить серии экспериментов в едином формате.
- Быстро строить графики по каналам (температура/давление/мощность/напряжение и т.д.).
- Получать сводные метрики (min/max/avg, длительность, число точек, пропуски/аномалии).
- Ускорить разбор результатов и сравнение экспериментов.

## Возможности MVP
- Создание эксперимента (метаданные: название, стенд, оператор, заметки).
- Импорт данных в эксперимент (CSV/JSONL).
- Просмотр списка каналов и их статистики.
- График по выбранным каналам и диапазону времени.
- Summary по эксперименту.
- Экспорт отфильтрованной выборки в CSV.

## Не входит в MVP
- Real-time приём данных (стриминг).
- Алерты/уведомления.
- Сложная аналитика (детект аномалий ML).
- Многопользовательская авторизация/роли.
